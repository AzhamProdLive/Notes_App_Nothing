## Getting Started
1. Clone the repository
   
   ```
   git clone https://github.com/AzhamProdLive/Notes_App_Nothing.git
   ```
   
2. Navigate to the project directory

   ```
   cd Notes_APP
   ```
   
3. Install dependencies

   ```
   flutter pub get
   ```

4. Run the app
   ```
   flutter run
   ```
